# 喜茶喜帖上传器-WITH-GUI
## 背景
本项目基于[@BienBoy](https://github.com/BienBoy)的[heytea-diy项目](https://github.com/BienBoy/heytea-diy)制作，
在维持核心功能不变的情况下增加了图形界面，图片预处理和自动代理功能。  
对手残党相对友好  
本程序基于[Microsoft .NET Framework 4.8](https://dotnet.microsoft.com/zh-cn/download/dotnet-framework/net48)开发，若无法运行请自行安装。
## 功能介绍
喜茶的喜帖制作只能在小程序上完成，对画师和手残党很不友好，
本程序用简洁的图形界面完成了将图片转换为黑白二色图并上传，
所有人打开**ps|sai|procreate...** ，这是命令（bushi）
## 使用方法
1. 预备工作：准备手机（有微信），电脑（有微信），一张图片（最好是黑白线稿）
1. **关闭所有的杀毒软件！！！** 因为程序涉及敏感操作极其容易误报，作者用浮木担保没有病毒，请放心使用
1. 右键以管理员身份运行*heytea-diy-gui.exe* ，依次完成Step1、Step2和Step3
1. **不要关闭弹出的任何窗口** **不要关闭弹出的任何窗口** **不要关闭弹出的任何窗口**
1. 使用后有小概率出现网络连接不上的情况，运行*clean.ps1*
